Personal Finance Dashboard

An Interactive Tableau Visualization

By Triet Nguyen

An interactive Tableau dashboard built to visualize personal finance data including income, expenses, and spending patterns by category. This project connects directly to the Budget Tracker Excel project, demonstrating a full data pipeline from data entry to visual analytics.

Dashboard Features
- Spending by Category — bar chart showing total income and expenses broken down by category
- Transaction Overview — individual transaction view color coded by category showing every income and expense
- Expense Breakdown — pie chart showing the proportion of spending across expense categories (housing, food, transport, utilities, entertainment)


Key Insights
- Housing accounts for the largest share of expenses at 70% of total spending
- Total income of $3,000 vs total expenses of $1,715 results in net savings of $1,285
- Food is the second largest expense category at 18% of total spending


Tools Used
- Tableau Public — dashboard creation and visualization
- Microsoft Excel — data source (Budget Tracker)
- GitHub — version control and portfolio hosting
